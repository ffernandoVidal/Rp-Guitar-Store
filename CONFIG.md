## Estructura del Proyecto

### Archivos Principales
- `index.html` - Página principal
- `menu.js` - Menú de navegación centralizado
- `footer.js` - Footer centralizado
- `nav.css` - Estilos de navegación
- `styles.css` - Estilos generales
- `productos.css` - Estilos para páginas de productos

### Componentes
- `components/cart.js` - Sistema de carrito de compras
- `components/cart.css` - Estilos del carrito
- `components/cart-templates.html` - Plantillas y ejemplos del carrito

### Secciones
- `guitarras/` - Guitarras eléctricas y acústicas
- `pedales/` - Pedales de efectos de múltiples marcas
- `amplificadores/` - Amplificadores
- `bajos/` - Bajos eléctricos
- `accesorios/` - Accesorios musicales
- `marcas/` - Marcas especiales
- `suhr/` - Productos Suhr
- `rp-music-school/` - Escuela de música

## Próximos Pasos para Producción

1. **Actualizar Información de Contacto**
   - Cambiar número de teléfono en footer.js
   - Cambiar email en footer.js
   - Actualizar links de redes sociales en footer.js

2. **Configurar WhatsApp Business**
   - Actualizar número en components/cart.js
   - Probar el envío de mensajes

3. **Agregar Imágenes Reales**
   - Reemplazar placeholders con fotos de productos reales
   - Optimizar imágenes para web

4. **SEO**
   - Agregar meta tags a todas las páginas
   - Crear archivo sitemap.xml
   - Agregar Google Analytics (opcional)

5. **Hosting**
   - Subir archivos a servidor web
   - Configurar dominio
   - Configurar SSL (HTTPS)

## Mantenimiento

### Para Actualizar el Menú
Editar el archivo `menu.js` y los cambios se aplicarán automáticamente a todas las páginas.

### Para Actualizar el Footer
Editar el archivo `footer.js` y los cambios se aplicarán automáticamente a todas las páginas.

### Para Agregar Nuevos Productos
1. Copiar estructura de producto existente
2. Actualizar información (nombre, precio, imagen, especificaciones)
3. El carrito funcionará automáticamente
